Technologies Used
HTML
CSS
JavaScript
Axios (for API requests)

Features
Search Functionality: Users can type in the search box and press "Enter" to display results in a table format.

Keyboard Shortcut: Pressing CTRL/CMD + / focuses the search box for quick access, enhancing user experience.

Dynamic Results Display: Results are shown in a paginated table, with the option to customize the number of results per page (1-10).

Loading Indicator: A loading spinner is displayed while data is being fetched, providing visual feedback during API calls.

Error Handling: The application handles various scenarios, including displaying messages for no results found or too many requests. 

Usage

Type in the search box to find places.

Use the pagination buttons to navigate through search results.

Adjust the results per page to customize the display.
